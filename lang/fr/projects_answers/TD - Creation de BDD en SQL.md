## Corrigé

### Voici la liste des commandes qu'il fallait effectuer dans l'ordre

- ```CREATE DATABASE Toto```
- ```CREATE DATABASE Tata```
- ```DROP DATABASE Toto```
- ```CREATE DATABASE toto```
- ```CREATE DATABASE IF NOT EXISTS toto``` ; le "IF NOT EXISTS" permet de ne créer la BDD que si celle-ci n'existe pas déjà.
- ```DROP DATABASE Tata```
- ```DROP DATABASE toto```
